# 🚀 내 Obsidian 시작 가이드

> 유튜브 최애 영상 3개의 핵심을 적용한 나만의 볼트입니다.

---

## 📁 폴더 구조

| 폴더 | 용도 |
|------|------|
| `00-Inbox/` | 아직 정리 안 된 메모 → 여기 먼저 |
| `10-Notes/` | 주제별 정리된 노트 |
| `20-Resources/` | 참고자료, 링크, 북마크 |
| `30-Daily/` | 날짜별 데일리 노트 |
| `99-Templates/` | 템플릿 (직접 작성 X) |
| `LLM-Wiki/` | AI가 자동 업데이트하는 지식 베이스 |

---

## ⚡ 지금 당장 할 3가지

1. **데일리 노트 시작** — `30-Daily/` 에 오늘 날짜 노트 만들기  
   템플릿: `99-Templates/데일리-노트-템플릿.md`

2. **링크 습관** — 노트에서 `[[다른노트]]` 걸기 연습

3. **웹 매뉴얼 보기** — ngrok URL로 접속하면 전체 가이드 보임

## 💡 핵심 원칙 (3영상 공통)

> "일단 쓰고, 나중에 정리. 링크가 폴더보다 중요하다."

- 완벽한 구조보다 **지속**이 중요
- 노트 1개 = 아이디어 1개
- `[[링크]]` 를 많이 걸수록 볼트가 살아있어짐
- 그래프 뷰 (`Ctrl+G`) 로 연결 확인
